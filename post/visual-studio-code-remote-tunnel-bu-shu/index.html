Visual Studio Code Remote Tunnel 部署

2022-12-14 2 min read # Tools

VS Code 更新到 1.74 后，推出了此项预览功能，提供了内网穿透的能力，使得可以在任何地方都访问到开启了此功能的设备。

可以通过 VS Code 连接到开启了远程隧道的设备，也可以通过浏览器进行访问。

通过 UI 进行设置


只需要登录账户之后 打开远程隧道访问... 即可，关闭了之后也就无法进行访问了，在开启后会有弹出一条通知可以复制链接，直接在浏览器中访问即可以连接到此设备。

通过 CLI 进行设置

安装

Ubuntu 上可以使用 snap 进行安装 sudo snap install code --classic，然而目前会存在问题，等修复后可以直接使用这种方式进行安装。

从这里, 手动下载 .deb 包，或者 cli 程序，进行安装。

安装完成后首先使用 code tunnel --accept-server-license-terms，根据提示进行登录，登录完成后，ctrl + c 结束程序，使用 code tunnel service install 保持程序长期在后台进行。code tunnel service uninstall 可以卸载此后台服务。

访问

Web

在vscode.dev，登录自己账户后可在侧栏中，远程资源管理器 中进行访问。

VS Code

安装插件 Remote - Tunnels，就可以在侧栏 远程资源管理器 中进行访问。
